# 음색 배정 가이드

## 배정 원칙

1. **성별 매칭**: 남성 캐릭터에는 남성 음성, 여성 캐릭터에는 여성 음성을 우선 고려합니다.
2. **나이 매칭**: 소년, 청년, 중년, 노년의 연령감에 맞는 음색을 고릅니다.
3. **성격 매칭**:
   - 활발하고 밝음 → 밝고 에너지 있는 음색
   - 차분하고 내성적임 → 낮고 안정적인 음색
   - 다정하고 섬세함 → 부드럽고 달콤한 음색
   - 위압적이고 강함 → 묵직하고 힘 있는 음색
4. **역할 매칭**: 주연은 식별성이 높은 음색을, 조연은 장면에 잘 섞이는 음색을 우선합니다.

## 작업 순서

1. `list_voices`로 사용 가능한 음색 목록을 확인합니다.
2. `get_characters`로 모든 캐릭터 정보를 가져옵니다.
3. 각 캐릭터의 성격, 나이, 성별, 역할을 분석합니다.
4. 각 캐릭터에 `assign_voice`를 호출해 적절한 음색을 배정합니다.
5. 배정 결과와 이유를 사용자에게 요약합니다.
