# 캐릭터와 장면 추출 가이드

## 캐릭터 추출 규칙

추출하는 캐릭터 정보는 다음을 포함합니다.

- **이름**: 캐릭터 전체 이름
- **역할**: 주연, 조연, 단역 등
- **외형 묘사**: 성별, 나이, 체형, 얼굴 특징, 헤어스타일, 의상 등
- **성격 특징**: 핵심 성격 태그
- **캐릭터 설명**: 배경 이야기와 관계

## 장면 추출 규칙

추출하는 장면 정보는 다음을 포함합니다.

- **장소**: 구체적인 공간 이름
- **시간**: 시간대와 빛 조건
- **분위기**: 환경 분위기 설명
- **프롬프트**: AI 이미지 생성에 사용할 영어 프롬프트. 배경 중심으로 작성하고 인물은 포함하지 않습니다.

## 소품 추출 규칙

추출하는 소품 정보는 다음을 포함합니다.

- **이름**: 소품명
- **유형**: 일상, 무기, 교통, 장식 등
- **설명**: 외형과 용도
- **이미지 프롬프트**: AI 이미지 생성용 영어 프롬프트

## 작업 순서

1. `read_script_for_extraction`으로 현재 회차 극본을 읽습니다.
2. `read_existing_characters`로 프로젝트의 기존 캐릭터와 현재 회차 연결 캐릭터를 확인합니다.
3. `read_existing_scenes`로 프로젝트의 기존 장면과 현재 회차 연결 장면을 확인합니다.
4. 현재 회차에 실제로 필요한 캐릭터와 장면만 추출합니다.
5. `save_dedup_characters`로 캐릭터를 저장하고 현재 회차에 연결합니다.
6. `save_dedup_scenes`로 장면을 저장하고 현재 회차에 연결합니다.

## 현재 회차 기준

- 목표는 현재 회차에 필요한 캐릭터와 장면을 보강하는 것입니다. 전체 프로젝트를 다시 훑는 작업이 아닙니다.
- 기존 캐릭터나 장면이 현재 회차에 연결되지 않았다면 재사용하고 현재 회차에 연결합니다.
- 같은 이름의 캐릭터나 같은 장소/시간대의 장면이 있으면 우선 재사용하고 중복 생성하지 않습니다.
