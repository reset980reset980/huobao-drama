# 극본 수정 가이드

## 수정 원칙

1. **핵심 줄거리 유지**: 주요 사건과 캐릭터 관계를 바꾸지 않습니다.
2. **시각성 강화**: 설명문을 장면과 행동 중심의 묘사로 바꿉니다.
3. **대사 중심 전개**: 가능한 한 대사로 사건을 밀고 나가며 과도한 내레이션을 줄입니다.
4. **짧은 호흡**: 각 장면은 30-60초 분량으로 구성합니다.
5. **카메라 언어 제외**: 샷 크기, 각도, 카메라 움직임은 스토리보드 단계에서 다룹니다.

## 형식화된 극본 형식

```markdown
## S01 | 실내 · 카페 | 황혼

황혼빛이 통창을 지나 카페 안으로 들어오고, 바 위의 커피잔에서 김이 오른다.

민수는 구석 좌석에 혼자 앉아 휴대폰을 내려다보며 불안한 표정을 짓는다.

문 종이 울리고 지연이 들어온다. 지연은 민수를 발견하고 미소를 지으며 다가간다.

지연: (미소) 오래 기다렸어?
민수: (고개를 든다) 아니, 방금 왔어.
```

## 형식 규칙

- `## S번호 | 실내/실외 · 장소 | 시간대` 형식의 장면 헤더를 사용합니다.
- 행동 묘사는 자연스러운 문단으로 작성하고 카메라 언어를 넣지 않습니다.
- 대사는 `캐릭터명: (상태/표정) 대사 내용` 형식으로 씁니다.

## 작업 순서

1. `read_episode_script`로 원본 내용을 읽습니다.
2. 대사, 설명, 심리 묘사의 비율과 장면 구조를 파악합니다.
3. `rewrite_to_screenplay`로 형식화된 극본을 작성합니다.
4. 형식과 분량이 맞는지 확인합니다.
5. `save_script`로 최종 결과를 저장합니다.

## 주의사항

- 심리 묘사는 표정, 행동, 오프 보이스로 바꿀 수 있습니다.
- 긴 설명문은 여러 짧은 장면으로 나눕니다.
- 각 장면에는 명확한 감정 전환점이 있어야 합니다.
- 캐릭터별 말투는 일관되게 유지합니다.
- 장면 번호는 S01, S02, S03처럼 연속되게 증가합니다.
- 시간대는 황혼, 심야, 새벽처럼 구체적으로 씁니다.
